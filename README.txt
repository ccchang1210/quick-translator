  HOTKEY 快捷鍵
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Ctrl+C  連按兩次    選取文字後連按兩次，自動帶入翻譯視窗
  Ctrl+X              開啟框選模式，用滑鼠框選畫面文字，OCR 後帶入翻譯視窗
  Esc                 關閉翻譯視窗 / 取消框選模式

  ── 翻譯視窗內 ──────────────────────────────────────────────
  [英→中] / [中→英]  切換翻譯方向
  [⇄]                 左右互換（交換輸入與結果）
  [🔊]                朗讀英文（再按一次 ⏹ 停止）
  [清除]              清空所有文字
  [複製譯文]          複製右側翻譯結果到剪貼簿

  一、首次安裝（只需做一次）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  步驟 1  安裝 Python 套件
  ─────────────────────────
    pip install -r requirements.txt


  步驟 2  安裝 Tesseract-OCR（框選 OCR 功能需要）
  ─────────────────────────────────────────────────
    下載網址：https://github.com/UB-Mannheim/tesseract/wiki
    （下載 tesseract-ocr-w64-setup-*.exe，Windows 64 位元版本）

    安裝時注意事項：
      ✦ 安裝路徑建議保持預設 C:\Program Files\Tesseract-OCR\
      ✦ 安裝頁面勾選「Additional language data (download)」
          → 找到並勾選「Chinese (Traditional)」（繁體中文）
          → 找到並勾選「Chinese (Simplified)」（簡體中文，選用）

    安裝完成後，將以下路徑加入系統 PATH 環境變數：
      C:\Program Files\Tesseract-OCR\

    或者，在 screen_capture.py 第 14 行取消註解並填入正確路徑：
      pytesseract.pytesseract.tesseract_cmd = r'C:\Program Files\Tesseract-OCR\tesseract.exe'


  步驟 3  語音朗讀（edge-tts，已在步驟 1 一併安裝）
  ──────────────────────────────────────────────────
    使用 Microsoft Edge 神經語音引擎，需要網路連線。
    首次按下 🔊 時會稍慢（~1 秒），之後正常。
    若無網路，自動退回 Windows 內建 SAPI 語音。

  二、啟動方式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  方法 A  直接雙擊
  ────────────────
    雙擊 start.bat

  方法 B  命令列
  ──────────────
    cd D:\claude\quick_translator
    python main.py

  啟動後程式會在背景執行（終端視窗保持開著）。
  要結束程式：在終端視窗按 Ctrl+C 一次，或直接關掉終端視窗。

  三、使用流程範例
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ▸ 翻譯網頁 / 文件中的英文
    1. 用滑鼠選取英文文字
    2. 按 Ctrl+C（複製）
    3. 再按一次 Ctrl+C（觸發翻譯）
    4. 翻譯視窗彈出，中文翻譯自動出現在右側

  ▸ 翻譯畫面上的文字（例如圖片、軟體介面、截圖）
    1. 按 Ctrl+X
    2. 用滑鼠拖曳框選要翻譯的區域
    3. 放開滑鼠，OCR 自動辨識，帶入翻譯視窗

  ▸ 查英文單字發音
    1. 確保翻譯視窗顯示英文在左側
    2. 按 🔊 按鈕，即可聽到 Microsoft 神經語音朗讀

  四、語音設定（更換聲音）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  編輯 tts_engine.py 第 46 行，將 VOICE 改為喜好的聲音：

    VOICE = 'en-US-JennyNeural'    ← 美式自然女聲（預設）
    VOICE = 'en-US-GuyNeural'      ← 美式自然男聲
    VOICE = 'en-GB-SoniaNeural'    ← 英式自然女聲
    VOICE = 'en-US-AriaNeural'     ← 表情豐富女聲
    VOICE = 'en-AU-NatashaNeural'  ← 澳洲女聲

  五、常見問題
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Q: 按 Ctrl+X 框選後顯示「需要安裝 Tesseract-OCR」
  A: 依「步驟 2」安裝 Tesseract，並確認已加入 PATH。

  Q: OCR 辨識英文正確，但中文亂碼
  A: Tesseract 安裝時未勾選 chi_tra 語言包。
     重新執行安裝程式，或手動下載 chi_tra.traineddata 放入：
     C:\Program Files\Tesseract-OCR\tessdata\

  Q: 翻譯視窗跑到後面，看不到
  A: 再按一次 Ctrl+C × 2 或 Ctrl+X，視窗會強制推到前景。

  Q: 朗讀按鈕沒有聲音
  A: 確認電腦有連接網路（edge-tts 需要網路）。
     若仍無聲，程式會自動切換為 Windows 內建 SAPI 語音。

  Q: 翻譯結果不準確
  A: 本工具使用 Google 翻譯（免費版），長句翻譯品質有限。
     翻譯需要網路連線。

  Q: 框選後 OCR 辨識率差
  A: - 確保框選範圍文字夠大（建議字體 > 12px）
     - 避免框選背景複雜的區域，純文字區域辨識率最高
     - 截圖中的文字若有旋轉或藝術字體，辨識率會降低

  七、開機自動啟動（選用）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. 按 Win+R，輸入 shell:startup，按 Enter
  2. 將 start.bat 的捷徑複製到開啟的資料夾中
  3. 之後每次開機會自動在背景啟動
